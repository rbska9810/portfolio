SKILL

HTML

웹 표준과 웹 접근성을 준수하는 HTML 정보 구조설계로 웹사이트 퍼블리싱을 진행합니다.

웹 표준을 위해 HTML에서 권장하는 마크업 방식과 시맨틱 태그를 사용해 전체 레이아웃 설계합니다.

클래스 네임과 함께 HTML 정보 구조설계를 꼼꼼히 작업한 후 퍼블리싱 작업을 진행합니다.

CSS

CSS3에서 제공하는 새로운 속성을 적극적으로 활용하여 다양하고 현대적인 디자인을 적용합니다.

디자인한 것을 완벽하게 구현할 수 있는 능력을 보유하고 있습니다.

소통을 통해 시각적 디자인을 코드로 정확하게 전환할 수 있습니다.

SASS

SCSS의 문법과 모듈화를 잘 활용하여 스타일 작업을 원활히할 수 있습니다.

JavaScript

Navigation, Tab Menu, Accordion 등의 상호작용을 구현하기 위해 JavaScript를 사용합니다.

플러그인 의존 없이 Vanilla-JavaScript 코딩을 통해 프로젝트에 필요한 기능들을 정확하게 구현합니다.

코드의 효율성을 높이기 위해 모듈화와 최적화 기술을 적극 활용하며, 코드의 가독성과 유지보수성을 고려하여 직관적이고 효과적인 코드를 작성합니다.

React

React를 사용하여 동적이고 효율적인 웹 애플리케이션을 개발한 경험이 있습니다.

React의 핵심 개념 중 하나인 컴포넌트를 통해 UI를 모듈화하고 재사용 가능한 구성 요소를 개발할 수 있습니다.

React Router를 활용하여 다양한 페이지 간의 라우팅을 구현할 수 있습니다.

React Hooks를 활용하여 함수형 컴포넌트에서 상태와 생명주기를 관리할 수 있습니다.

React를 이용하여 프로젝트를 구현함으로써, 확장 가능하고 유지보수가 용이한 웹 애플리케이션을 개발할 수 있는 능력을 갖추고 있습니다.

BootStrap

부트스트랩 프레임워크를 사용하여 웹페이지를 빠르고 쉽게 개발할 수 있습니다.

부트스트랩을 사용한 반응형 디자인, 그리드시스템 플러그인 등의 기능을 구현 할 수 있습니다.

Figma

피그마를 통한 디자인, 프로토타이핑, 협업 등의 다양한 기능을 숙련도 있게 활용합니다.

기기의 사이즈와 특성을 고려하여 모바일, 태블릿, 데스크탑 등 다양한 디바이스에 최적화된 UI/UX 디자인을 제작합니다.

피그마를 활용하여 프로토타입을 제작하고 사용자 피드백을 수용하여 디자인을 지속적으로 개선합니다.

GitHub, Git

소스 코드의 버전 관리 및 협업을 위해 Git을 사용하며, GitHub를 통해 프로젝트를 호스팅하고 협업을 해본 경험이 있습니다.

Slack

Slack을 사용하여 팀원과 협업할 수 있습니다.
